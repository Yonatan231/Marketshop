# proyecto_pagina_web_MFMS

Aplicación web de e-commerce para maquillaje y zapatos construida con Python y Flask.

## Características

- **Estilo visual**: tonos rosados, nude, blanco y dorado con apariencia tipo app móvil.
- **Responsive**: diseñado para móvil y escritorio.
- **Secciones**:
  - Pantalla de login con inicio social
  - Inicio con banner promocional y productos destacados
  - Detalle de producto con precio, descuento y calificación
  - Carrito con total, cupón y checkout
  - Recompensas gamificadas: check-in diario y ruleta de premios
- **Funcionalidades**: agregar al carrito, aplicar descuentos, experiencia tipo aplicación móvil.

## Tecnologías

- Python 3
- Flask
- HTML
- CSS
- JavaScript

## Estructura del proyecto

- `app.py` - servidor Flask
- `templates/index.html` - plantilla principal
- `static/styles.css` - estilos
- `static/script.js` - lógica de interfaz
- `requirements.txt` - dependencias

## Cómo ejecutar

1. Instala dependencias:

```bash
pip install -r requirements.txt
```

2. Inicia la aplicación:

```bash
python app.py
```

3. Abre el navegador en:

```bash
http://127.0.0.1:5000
```
